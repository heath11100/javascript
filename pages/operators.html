Operators

Addition (+)

The addition operator is used to add two values together.

Numbers

The addition operator is most commonly used to add two number values together. 

        let number1 = 5;
        let number2 = 10;
        let number3 = number1 + number2;
        // number3 = 15

Subtraction (-)

The addition operator is used to subtract a number from another number. 

        let number1 = 5;
        let number2 = 10;
        let number3 = number1 - number2;
        // number3 = -5

Multiplication (*)

The multiplication operator is used to multiply two numbers together. 

        let number1 = 5;
        let number2 = 10;
        let number3 = number1 * number2;
        // number3 = 50

Division (/)

The multiplication operator is used to divide one number by another number. 

        let number1 = 5;
        let number2 = 10;
        let number3 = number1 / number2;
        // number3 = 0.5
